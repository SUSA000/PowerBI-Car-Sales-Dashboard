# 🚗 Car Sales Power BI Dashboard

## 📌 Project Overview
This project involves the design and development of a dynamic and interactive Car Sales Dashboard using Power BI. The dashboard visualizes critical KPIs related to car sales, helping to understand sales performance over time and make data-driven decisions.

## 🎯 Problem Statements Addressed
**1. KPI Requirements:**
* Year-to-Date (YTD) & Month-to-Date (MTD) Total Sales
* YTD & MTD Average Price
* YTD & MTD Cars Sold
* Year-over-Year (YOY) Growth metrics

**2. Chart Requirements:**
* YTD Sales Weekly Trend (Line Chart)
* YTD Total Sales by Body Style & Color (Donut Charts)
* YTD Cars Sold by Dealer Region (Map Visual)
* Company-Wise Sales Trend (Matrix with Data Bars)

## 🛠️ Tools & Technologies Used
* **Power BI:** Data visualization, DAX measures, Data modeling.
* **Excel:** Raw dataset storage.
